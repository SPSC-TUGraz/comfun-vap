# comfun-vap
comfun-vap predicts fine-grained communicative functions of speaker dyads from conversational audio using Voice Activity Projection.
## Status: Pre-release
This repository accompanies the paper "Communicative Function Classification using Voice Activity Projection" (Paierl M., Schuppler B., TSD 2026).
The complete code, trained models, and reproduction instructions will be made fully public here upon publication. The repository is currently being prepared.
